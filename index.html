SKILLS
  • React

    동적 웹 애플리케이션 개발 경험이 있으며, 컴포넌트 기반 아키텍처와
    상태 관리 라이브러리를 통해 효율적인 데이터 흐름을 구현했습니다.

    Hooks, JSX, 리액트 라우터, Axios를 사용하여 함수형 컴포넌트 작성과
    API 통신 경험을 보유하고 있습니다.
  • JavaScript

    JavaScript를 활용한 동적 웹 애플리케이션 개발 경험이 있으며,
    ES6+ 문법과 AJAX를 통해 비동기 데이터 처리 및 API 통신을 수행했습니다.

    또한, DOM 조작, 함수형 프로그래밍, JavaScript 라이브러리 활용, 모듈화 및
    패키지 관리의 경험도 갖추고 있습니다.
  • PWA

    PWA를 활용하여 오프라인 지원과 빠른 로딩 경험을 제공하며,
    서비스 워커로 캐싱과 데이터 동기화를 구현했습니다.

    이를 통해 웹 앱 매니페스트를 사용하여 홈 화면 추가와
    푸시 알림 기능으로 사용자 참여를 유도할 수 있는 역량을 갖추고 있습니다.
  • SCSS

    SCSS를 활용하여 효율적인 스타일링과 코드 관리를 수행한 경험이 있으며,
    중첩 기능과 믹스인으로 가독성 높은 스타일을 작성했습니다.

    또한, 변수와 상속을 통해 일관된 스타일을 유지하고,
    반응형 웹 디자인 구현에도 참여한 경험이 있습니다.
  • Tailwind CSS

    유틸리티 클래스 기반의 Tailwind CSS를 활용하여 스타일링과 재사용 가능한
    컴포넌트 조합을 통해 효율적으로 작업할 수 있는 기술을 갖추고 있습니다.

    또한, Tailwind CSS의 커스터마이징 기능과 반응형 디자인 구현을 통해
    일관된 스타일을 유지할 수 있는 기술을 보유하고 있습니다.
